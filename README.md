# IP Information Grabber

IP Information Grabber is a powerful and user-friendly Python bot that provides detailed information about IP addresses. With this bot, you can easily retrieve data such as the country, region, city, ZIP code, latitude, and longitude associated with an IP address.

## Features

- Get information about your own IP address or any targeted IP address.
- Retrieve data such as country, region, city, ZIP code, latitude, and longitude.
- Utilizes the ip-api.com API to fetch accurate and up-to-date information.

## How to Use

1. Clone the repository and navigate to the project directory.
2. Install the required dependencies using `pip install -r requirements.txt`.
3. Run the Python script `ip_info_bot.py`.
4. Choose whether you want to retrieve information about your own IP address or a targeted IP address.
5. Follow the on-screen prompts to input the IP address (if applicable).
6. Sit back and let the IP Information Grabber fetch the data for you!
